# Task Manager API

This is a backend project I’m building while learning backend development and preparing for software roles.

The goal of thre project is to build a simple but production style API where users can create and manage tasks with proper authentication.

## Whats done so far
- Basic Flask app setup
- Project structure with config and models
- Health check and base routes

## Tech Stack
- Python
- Flask
- SQLAlchemy
- JWT (pending)
